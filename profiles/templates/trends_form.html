Trends will be shown for the selected energy and for the available data in the scv file.

  • Enter desired energy. The energy must be one of those that are listed in the csv file (and in your mapcheck file names).
  • Press Select energy and Run Trends button.
  • Trends will be generated for that energy.
  • Press Plot Trends bitton repeatedly to see trends for other energies.
  • There is no need to run top three buttons if you already have a csv file from last time. Just select it in Prepare Trends step and run it.
  • Please note: trends will show only for those dates. machines and energies that are present in the csv file.
  • The csv file will have only the dates that have been accumulated in one session by running multiple dates in a row by pressing the Load Monthly button with subsequent clicking the Save CSV button.
