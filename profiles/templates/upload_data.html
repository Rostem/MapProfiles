Upload monthly mapcheck files:

  • Press Choose Files, the File Upload window will open. Navigate to your locally saved mapcheck files, select those that belong to the chosen machine and date,
  • Wait until "No file chosen" changes to a number (of loading files).
  • Then press Load Monthly ... button and wait a few seconds for files to finish uploading.
  • On completion of file upload, the program will process data and show results.
